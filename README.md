🕐 Digital Clock
A sleek digital clock built with HTML, CSS, and JavaScript — featuring a retro 7-segment display font, live time, date, and a day-of-week indicator.

✨ Features

Live time display (Hours : Minutes : Seconds)
Date, Month, and Year display
Day-of-week indicator with a highlight dot
Retro digital font (DSEG7 Classic)


🛠️ Tech Stack

HTML5
CSS3
Vanilla JavaScript

├── digital_clock.js         # Clock logic
